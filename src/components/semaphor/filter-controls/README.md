# Semaphor Filter Controls

Filter controls for Semaphor Data App input handles.

This block includes:

- SDK-backed single-select and multi-select controls;
- date range picker and Semaphor date-range filter binding;
- active filter summary badge;
- option adapter utilities for preserving Semaphor option values.

The controls consume public `react-semaphor/data-app-sdk` input handles and do
not infer analytics semantics from labels or row values.
